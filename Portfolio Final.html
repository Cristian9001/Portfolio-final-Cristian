Sobre mí:

Proyectos:

Proyecto Google:

En este proyecto tuve que crear el menú principal de Google.

Para ello usé divs para poder separar las cosas, imágenes y botones, todo para el HTML.

En el caso del CSS, en las imágenes, usé los gap, los margin, el align-items, el justify-content y más cosas para poder cambiar la posición de las imágenes.

También cambié el color del fondo usando el background-color en el CSS.


Proyecto MySQL:

En este proyecto usé la aplicación de MySQL Workbench para poder crear bases de datos.

Como podemos ver en la parte de la izquierda de la imagen, hay varias bases de datos hechas en el programa.


Proyecto YouTube:

Proyecto sobre crear la página principal de YouTube.

Para este proyecto, en el HTML, he usado sections para separar partes como la parte de navegación con el inicio, el historial, las suscripciones y más; también he usado divs para poder separar las partes del YouTube y poner enlaces de videos.


Proyecto Portfolio Ejemplo
